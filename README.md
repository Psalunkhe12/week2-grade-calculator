# Student Grade Calculator

---

## 📌 Project Overview

The Student Grade Calculator is a Python-based application designed to process multiple students' marks, calculate their grades, and provide class statistics.  

This project demonstrates the use of control flow, data structures, and error handling to build a real-world application.

---

## 🎯 Objectives

- To implement conditional logic using if/elif/else  
- To use lists for storing multiple student records  
- To apply loops for repeated operations  
- To validate user input using error handling  
- To organize code using functions  
- To calculate and display class statistics  

---

## 🛠️ Tools and Technologies Used

- Programming Language: Python  
- Editor: VS Code / Notepad  
- Platform: Windows
